Skripti jaoks vajalike andmete allalaadimise juhend:
* Mine lehele https://geoportaal.maaamet.ee/est/Ruumiandmed/Eesti-topograafia-andmekogu/Laadi-ETAK-andmed-alla-p609.html
* Lae alla Ehitiste SHP-fail (~120M)
* Ava allalaetud ETAK_Eesti_SHP_ehitised.zip 
* Paki lahti kõik E_401_hoone_ka-nimelised failid käesolevasse kausta
* Vajaduse korral uuenda config.py


Instructions to download data used by script:
* Go to https://geoportaal.maaamet.ee/eng/Maps-and-Data/Estonian-Topographic-Database/Download-Topographic-Data-p618.html
* Download Buildings SHP-file (~120M)
* Open downloaded ETAK_Eesti_SHP_ehitised.zip
* Extract all files named E_401_hoone_ka into this folder.
* If needed, update config.py
